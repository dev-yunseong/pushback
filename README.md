# Pushback

Pushback is an agent skill that challenges weak coding, architecture, product, and implementation decisions before they turn into code.

It checks reasoning, exposes hidden assumptions, identifies missing constraints, and asks for better-supported alternatives.

## Install

From a local checkout:

```sh
npx skills add . --skill pushback
```

After publishing to GitHub:

```sh
npx skills add dev-yunseong/pushback --skill pushback
```

Use it explicitly:

```text
$pushback Review this approach before I implement it.
```
